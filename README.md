## 城市地铁规划分析

#### 项目起因

找到一个模型来评价一个城市地铁路线图的优秀程度（北京地铁烂到爆炸啊啊啊啊）

#### 项目update

- First metric

把一个地铁网格图建立一个graph，计算每两个node（地铁站）之间坐地铁需要的最短路程，比上它们之间的直线路程

问题：

a. 没有很好的数据来建模两个地铁站之间的路程（目前是使用两个地铁站的经纬度来计算直线距离）

b. 换乘没有加权

c. 这个metric有可能会被一些郊区地区的地铁站大幅bias，以至于不能反映出核心地段的路线优良程度
